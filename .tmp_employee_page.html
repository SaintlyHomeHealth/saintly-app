Admin Guidance

1. Start from the current event

Admin staff should always enter forms through the employee dashboard so the correct annual event is used.

2. Draft before finalize

Draft status protects in-progress work. Finalize should only happen after the live review is fully complete and survey-safe.

3. Preserve annual history

Each annual cycle should create a fresh event so prior-year performance and competency records stay intact for CHAP and audit review.
